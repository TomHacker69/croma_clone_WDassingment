# 🚀 Latest Gadgets Website

A simple and modern **Gadget Showcase Website** built using HTML and CSS.
This project displays the latest technology products like laptops, smartphones, and tablets in a clean and structured layout.

---

## 📌 Features

* 🖥️ Clean and minimal UI design
* 📱 Responsive layout (mobile-friendly)
* 📋 Organized product categories
* 📊 Product comparison table
* 🌐 Footer with useful links
* ✉️ Contact email integration

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
