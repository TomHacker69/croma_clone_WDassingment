# 🚀 Latest Gadgets Website

A modern and responsive **Gadget Showcase Website** built using HTML and CSS.
This project presents the latest technology products like laptops, smartphones, and tablets in a visually appealing UI.

---

## 📌 Features

* 🖥️ Clean and modern UI design
* 📱 Responsive layout (works on mobile & desktop)
* 🧩 Card-based product sections
* 📊 Product comparison table
* 🌐 Footer with useful links and social media
* ✉️ Contact email integration

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3** (Flexbox + Grid + Animations)
